View Footprint In:   ArcGIS.com Map

Service Description: The Australian Topographic base map service is seamless national dataset coverage for the whole of Australia. The map is a representation of the Geoscience Australia 250k topographic specification and portrays a detailed graphic representation of features that appear on the Earth's surface. These features include cultural, hydrography and relief themes. The service contains layer scale dependencies.

Map Name: Australian Topographic Base Map (Web Mercator)
